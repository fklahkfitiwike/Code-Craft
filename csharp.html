Code Craft

Overview and Introduction

C# (pronounced "C-sharp") is a modern, object-oriented programming language developed by Microsoft. It is widely used for building Windows applications, web applications, and game development with Unity. C# is part of the .NET ecosystem.

Setup and Installation

To get started with C#, you'll need to install Visual Studio, a powerful Integrated Development Environment (IDE) for .NET development. You can download it from Microsoft Visual Studio.

Alternatively, you can use Visual Studio Code with the C# extension for a lightweight coding environment.

Basic Syntax and Concepts

Variables:

int x = 5;
string y = "Hello, World!";

Data Types: int, float, string, bool, char

Data Type Description Example
int Integer int number = 10;
float Floating-point number float temperature = 36.6F;
string String string name = "Alice";
bool Boolean bool isActive = true;
char Character char letter = 'A';

Control Structures: if, for, while

if (x > 0)
{
    Console.WriteLine("Positive");
}
else
{
    Console.WriteLine("Non-positive");
}

for (int i = 0; i < 5; i++)
{
    Console.WriteLine(i);
}

int j = 0;
while (j < 5)
{
    Console.WriteLine(j);
    j++;
}

Functions:

public string Greet(string name)
{
    return "Hello, " + name;
}

Console.WriteLine(Greet("Alice"));

Advanced Concepts

Classes and Objects:

public class Person
{
    public string Name { get; set; }
    public int Age { get; set; }

    public string GetDetails()
    {
        return $"Name: {Name}, Age: {Age}";
    }
}

Person p = new Person() { Name = "Alice", Age = 25 };
Console.WriteLine(p.GetDetails());

Exception Handling:

try
{
    int result = 10 / 0;
}
catch (DivideByZeroException ex)
{
    Console.WriteLine("Cannot divide by zero!");
}

File I/O:

using System.IO;

string text = File.ReadAllText(@"C:\example.txt");
Console.WriteLine(text);

File.WriteAllText(@"C:\example.txt", "Hello, World!");

Practical Examples and Projects

Simple Console Application:

using System;

namespace HelloWorld
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Hello, World!");
        }
    }
}

ASP.NET Web Application:

using Microsoft.AspNetCore.Builder;
using Microsoft.AspNetCore.Hosting;
using Microsoft.Extensions.DependencyInjection;

public class Startup
{
    public void ConfigureServices(IServiceCollection services)
    {
        services.AddControllersWithViews();
    }

    public void Configure(IApplicationBuilder app, IWebHostEnvironment env)
    {
        app.UseRouting();
        app.UseEndpoints(endpoints =>
        {
            endpoints.MapControllerRoute(
                name: "default",
                pattern: "{controller=Home}/{action=Index}/{id?}");
        });
    }
}
