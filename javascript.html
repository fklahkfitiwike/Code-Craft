Overview and Introduction

JavaScript is a dynamic programming language primarily used for web development to create interactive effects within web browsers. It is an essential technology alongside HTML and CSS, enabling dynamic and interactive websites.

Setup and Installation

JavaScript is supported by most modern web browsers, which have built-in JavaScript engines. To get started, you can use a text editor or IDE such as Visual Studio Code, Sublime Text, or Atom to write your JavaScript code.

Embedding JavaScript in an HTML document:

  • Internal JavaScript: Include JavaScript code within the <script> tag inside the HTML document.
  • External JavaScript: Write JavaScript code in a separate file with a .js extension (e.g., script.js) and link it to the HTML document using the <script> tag.

<!-- Internal JavaScript -->
<script>
    console.log("Hello, World!");
</script>

<!-- External JavaScript -->
<script src="script.js"></script>

Basic Syntax and Concepts

Variables:

  • var, let, and const: Declare variables. var is function-scoped, while let and const are block-scoped.

var name = "John";
let age = 30;
const country = "USA";

Data Types:

  • String, Number, Boolean, Object, Array, Null, Undefined, Symbol

let str = "Hello, World!";
let num = 42;
let isActive = true;
let person = { firstName: "John", lastName: "Doe" };
let numbers = [1, 2, 3, 4, 5];
let empty = null;
let undef;

Operators: Arithmetic, Assignment, Comparison, Logical, etc.

let sum = 5 + 3;
let isEqual = (5 === 5);
let isBothTrue = (true && false);

Control Structures: if, for, while

if (age > 18) {
    console.log("Adult");
} else {
    console.log("Not an adult");
}

for (let i = 0; i < 5; i++) {
    console.log(i);
}

let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}

Functions: function keyword and arrow functions

function greet(name) {
    return "Hello, " + name;
}

let add = (a, b) => a + b;

Advanced Concepts

Object-Oriented Programming: Classes and Objects

class Person {
    constructor(firstName, lastName) {
        this.firstName = firstName;
        this.lastName = lastName;
    }

    getFullName() {
        return this.firstName + " " + this.lastName;
    }
}

let person = new Person("John", "Doe");
console.log(person.getFullName());

Promises and Async/Await: Handling asynchronous operations

let fetchData = new Promise((resolve, reject) => {
    let data = "Sample Data";
    resolve(data);
});

fetchData.then(data => {
    console.log(data);
}).catch(error => {
    console.log(error);
});

async function getData() {
    try {
        let data = await fetchData;
        console.log(data);
    } catch (error) {
        console.log(error);
    }
}
getData();

DOM Manipulation: Interacting with the HTML Document Object Model

document.getElementById("myElement").innerHTML = "New Content";
document.querySelector(".myClass").style.color = "blue";

Libraries and Frameworks

Popular libraries and frameworks for front-end development:

  • React: A JavaScript library for building user interfaces.
  • Angular: A platform for building mobile and desktop web applications.
  • Vue: A progressive framework for building user interfaces.

Server-side development with Node.js: A runtime environment that executes JavaScript code outside a web browser.

const express = require('express');
const app = express();

app.get('/', (req, res) => {
    res.send('Hello, World!');
});

app.listen(3000, () => {
    console.log('Server is running on port 3000');
});

Practical Examples and Projects

Interactive To-Do List App:

<!-- HTML -->
<input type="text" id="new-task" placeholder="Add a new task">
<button onclick="addTask()">Add Task</button>
<ul id="task-list"></ul>

<!-- JavaScript -->
<script>
    function addTask() {
        let task = document.getElementById("new-task").value;
        let listItem = document.createElement("li");
        listItem.textContent = task;
        document.getElementById("task-list").appendChild(listItem);
    }
</script>

Simple Game using HTML Canvas:

<!-- HTML -->
<canvas id="gameCanvas" width="500" height="400"></canvas>

<!-- JavaScript -->
<script>
    let canvas = document.getElementById("gameCanvas");
    let context = canvas.getContext("2d");

    context.fillStyle = "#FF0000";
    context.fillRect(0, 0, 50, 50);
</script>

RESTful API with Node.js:

const express = require('express');
const app = express();
const port = 3000;

let items = [];

app.use(express.json());

app.get('/items', (req, res) => {
    res.json(items);
});

app.post('/items', (req, res) => {
    let newItem = req.body;
    items.push(newItem);
    res.status(201).json(newItem);
});

app.listen(port, () => {
    console.log(`Server running at http://localhost:${port}/`);
});
