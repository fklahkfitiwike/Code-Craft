Overview and Introduction

R is a language and environment for statistical computing and graphics. It is highly extensible and provides a wide variety of statistical and graphical techniques.

Setup and Installation

To get started with R, you can use RStudio, a powerful IDE for R.

Setting up your development environment:

  • Install R from CRAN.
  • Download and install RStudio from RStudio.
  • Create a new R project in RStudio.

Basic Syntax and Concepts

Variables:

x <- 5
y <- "Hello, World!"

Data Types: numeric, character, logical, factor, data frame

Data Type Description Example
numeric Numeric values x <- 5
character Character strings y <- "Hello, World!"
logical Boolean values isTrue <- TRUE
factor Categorical data factor_var <- factor(c("low", "medium", "high"))
data frame Tabular data df <- data.frame(x=c(1,2,3), y=c("a","b","c"))

Control Structures: if, for, while

if (x > 0) {
    print("Positive number")
} else {
    print("Non-positive number")
}

for (i in 1:5) {
    print(i)
}

i <- 1
while (i <= 5) {
    print(i)
    i <- i + 1
}

Functions:

my_function <- function(name) {
    return(paste("Hello", name))
}

print(my_function("World"))

Advanced Concepts

Data Manipulation with dplyr:

library(dplyr)
df <- data.frame(x=c(1,2,3), y=c("a","b","c"))
df <- df %>% filter(x > 1) %>% mutate(z = x * 2)
print(df)

Data Visualization with ggplot2:

library(ggplot2)
ggplot(data=df, aes(x=x, y=z)) + geom_point()

Exception Handling

R uses the tryCatch function to handle exceptions gracefully.

result <- tryCatch({
    10 / 0
}, warning = function(w) {
    print("Warning occurred")
}, error = function(e) {
    print("Error occurred")
}, finally = {
    print("Completed")
})

Practical Examples and Projects

Simple Shiny App:

library(shiny)
ui <- fluidPage(
    titlePanel("Simple Shiny App"),
    sidebarLayout(
        sidebarPanel(
            sliderInput("num", "Choose a number", 1, 100, 50)
        ),
        mainPanel(
            textOutput("result")
        )
    )
)
server <- function(input, output) {
    output$result <- renderText({
        paste("You chose", input$num)
    })
}
shinyApp(ui = ui, server = server)
