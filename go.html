Overview and Introduction

Go, also known as Golang, is a statically typed, compiled programming language designed for simplicity and efficiency. It is widely used for backend development, cloud services, and system programming.

Setup and Installation

To get started with Go, you'll need to install the Go programming environment.

Setting up your development environment:

  • Download and install Go from golang.org.
  • Set up your workspace by creating a directory for your Go projects.
  • Add the Go binary to your system's PATH.

Advanced Concepts

Structs:

type Person struct {
    Name string
    Age  int
}

func (p Person) getDetails() string {
    return fmt.Sprintf("Name: %s, Age: %d", p.Name, p.Age)
}

p := Person{Name: "Alice", Age: 25}
fmt.Println(p.getDetails())

Interfaces:

type Describable interface {
    Describe() string
}

type Car struct {
    Make  string
    Model string
}

func (c Car) Describe() string {
    return fmt.Sprintf("Car: %s %s", c.Make, c.Model)
}

func describe(d Describable) {
    fmt.Println(d.Describe())
}

myCar := Car{Make: "Toyota", Model: "Corolla"}
describe(myCar)

Concurrency (Goroutines and Channels):

func printNumbers() {
    for i := 1; i <= 5; i++ {
        fmt.Println(i)
        time.Sleep(100 * time.Millisecond)
    }
}

func main() {
    go printNumbers()

    for i := 'A'; i <= 'E'; i++ {
        fmt.Printf("%c ", i)
        time.Sleep(150 * time.Millisecond)
    }

    fmt.Println("Main function finished")
}

Exception Handling

Go does not have exceptions, but uses multiple return values and error handling with the error type.

func divide(a, b float64) (float64, error) {
    if b == 0 {
        return 0, fmt.Errorf("cannot divide by zero")
    }
    return a / b, nil
}

result, err := divide(4, 2)
if err != nil {
    fmt.Println("Error:", err)
} else {
    fmt.Println("Result:", result)
}

Practical Examples and Projects

Simple Web Server:

package main

import (
    "fmt"
    "net/http"
)

func handler(w http.ResponseWriter, r *http.Request) {
    fmt.Fprintf(w, "Hello, World!")
}

func main() {
    http.HandleFunc("/", handler)
    fmt.Println("Starting server at :8080")
    http.ListenAndServe(":8080", nil)
}

Basic REST API with gorilla/mux:

package main

import (
    "encoding/json"
    "net/http"
    "github.com/gorilla/mux"
)

type Item struct {
    ID   string `json:"id"`
    Name string `json:"name"`
}

var items []Item

func getItems(w http.ResponseWriter, r *http.Request) {
    json.NewEncoder(w).Encode(items)
}

func main() {
    items = append(items, Item{ID: "1", Name: "Item One"})

    r := mux.NewRouter()
    r.HandleFunc("/items", getItems).Methods("GET")

    http.ListenAndServe(":8080", r)
}
